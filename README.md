# Arcade Top-Down Shooter

A fast-paced, single-file top-down shooter game built with **Pygame**. This project is a simple, asset-free game that focuses on dynamic gameplay mechanics and a clear, modern visual style.

---

## Features

* **Dynamic Gameplay**: Dodge and weave through hordes of enemies in a procedurally generated map.
* **Two Unique Weapons**:
    * **Rifle**: A fast-firing weapon for precision.
    * **Shotgun**: A powerful, wide-spread weapon for close-quarters combat.
* **Dash Mechanic**: Use your dash for a quick burst of speed and temporary invincibility, creating a knockback effect on enemies.
* **Visual Flair**: Experience a gritty feel with high-contrast screen shake, muzzle flashes, and sparks.
* **Clear UI**: Keep track of your ammo, health, score, and current wave with a simple and clean user interface.

---

## Controls

| Key | Action |
| :--- | :--- |
| **W, A, S, D** | Move |
| **Mouse** | Aim |
| **Left Click** | Shoot (hold) |
| **R** | Reload |
| **Q** | Switch Weapon |
| **Space** | Dash |
| **Esc** | Pause |
| **F11** | Toggle Fullscreen |
